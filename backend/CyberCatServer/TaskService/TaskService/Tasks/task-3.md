# Шкафчик с замком
На вход постоянно поступают 6 чисел.
Если в числах есть 1 число из пароля - выведи "Один".
Если в числах есть 2 и более чисел из пароля - выведи "Два".
Если в числах нету чисел из пароля - выведи "-"

**Пример 1:**

| Вход: `190246`
| Вывод: `Один`

**Пример 2:**

| Вход: `222222`
| Вывод: `-`
